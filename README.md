# Page Grabber for AI

抓取网页内容并复制到剪贴板，方便粘贴到任意 AI 对话中。支持智能正文提取、Markdown 格式化、Prompt 模板拼接、长文自动分段、跨页面多次收集。

## 安装

1. 打开 Chrome，地址栏输入 `chrome://extensions/`
2. 右上角开启「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本项目文件夹

## 基本用法

### 抓取全页面
- 点击浏览器右上角插件图标 → 弹窗中「全页面」标签 → 点击「复制全页面到剪贴板」
- 或按快捷键 `Cmd+Shift+G`
- 或点击页面右下角的浮动按钮

### 收集选中内容（跨页面累积）
- 在页面上选中一段文字 → 右键 →「收集选中内容」
- 可在不同页面重复操作，内容会持续累积，插件图标上显示已收集数量
- 也可用快捷键 `Cmd+Shift+S` 收集当前选中文字
- 点击插件图标 → 切换到「已收集」标签页 → 查看、编辑、复制所有收集的内容

### 带 Prompt 模板
- 在「全页面」或「已收集」标签页中，从下拉菜单选择模板（总结、翻译、提取关键信息等）
- 复制的内容会自动拼接模板前缀，粘贴到 AI 对话框即可

## 长文分段

当内容超过设定的 token 上限（默认 8000）时，自动按段落分割：
- 弹窗底部出现分段指示器 `第 1/N 部分`
- 用左右箭头切换部分，逐段复制粘贴
- 每段自动加前缀 `[第1/3部分] 请等我发完所有部分后再回复`

## 设置

右键插件图标 → 选项，或在 `chrome://extensions/` 点击插件的「详情 → 扩展程序选项」：

| 设置项 | 说明 |
|---|---|
| 输出格式 | Markdown（保留标题/列表/表格/代码块）或纯文本 |
| 每段最大 token 数 | 长文分段阈值，默认 8000 |
| 浮动按钮 | 开关页面右下角的快捷抓取按钮 |
| Prompt 模板管理 | 添加/删除自定义模板 |
| 清除历史 | 删除所有抓取记录 |

## 快捷键

| 快捷键 | 功能 |
|---|---|
| `Cmd+Shift+G` | 抓取全页面并复制 |
| `Cmd+Shift+S` | 收集当前选中内容 |

> Windows/Linux 上将 `Cmd` 替换为 `Ctrl`

## 弹窗标签页

| 标签 | 功能 |
|---|---|
| 全页面 | 提取当前页面正文，预览、选模板、复制 |
| 已收集 | 查看所有收集的片段，支持单条删除、全部清空、合并复制 |
| 历史 | 最近 100 次抓取记录，点击可重新打开对应页面 |
